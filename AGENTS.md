# PD — Покер с друзьями (Telegram Mini App)

Мини-апп для игры в Техасский Холдем с друзьями: любой создаёт стол, получает
код/ссылку-приглашение и зовёт друга. Реальное время через WebSocket (Socket.IO).

## Запуск
```
npm install
npm start            # PORT=3000 по умолчанию
```
Открыть `http://localhost:3000`. Без Telegram работает dev-режим (гость с
генерируемым id, имя в localStorage) — можно тестировать в браузере в двух вкладках.

С реальным Telegram: задать `BOT_TOKEN` в окружении — тогда сервер проверяет
подпись `initData` мини-аппа. Без `BOT_TOKEN` подпись не проверяется (dev).

## Структура
- `server.js` — HTTP (express, отдаёт `public/`) + Socket.IO. Авторизация
  (`verifyInitData` по HMAC BOT_TOKEN / dev-фолбэк), комнаты столов (Map).
  События: `hello`, `createTable`, `joinTable`, `startHand`, `action`,
  `leaveTable`. Рассылка `state` каждому игроку через `broadcast()`.
- `src/poker.js` — колода, `shuffle`, `evaluate7(cards)` — лучшая 5-карточная
  рука из 7 (перебор C(7,5)), `cmpScore` для сравнения. 9 категорий, учтено колесо A-5.
- `src/table.js` — класс `Table`: рассадка (6 мест), блайнды (в т.ч. heads-up),
  раздача, круги торговли, олл-ины с **сайд-потами** по `totalBet`, вскрытие,
  раздача банка. `stateFor(userId)` скрывает чужие карты до вскрытия.
- `public/` — `index.html` (лобби/создание/стол), `style.css` (premium-казино:
  шрифты Playfair Display + Sora, палитра латунь+фетр, стеклянные панели,
  анимации раздачи/подсветка хода, адаптив 390px), `app.js` (Socket.IO-клиент,
  рендер стола по эллипсу, слайдер рейза, кнопка «Пригласить» через `t.me/share`).

## Хостинг
Задеплоен на Render (free): `https://poker-dream.onrender.com/`. Код в GitHub
клиента `AlexeyTretyakov/youpoker` (push в `main` → авто-деплой). `render.yaml`
в корне. Mini App URL прописывается в @BotFather.

## Проверено
Движок оценки рук (10 категорий + сравнение) и полные раздачи через socket-клиентов:
колл→вскрытие, рейз, олл-ин с сайд-потом — фишки всегда сохраняются (сумма стеков
неизменна). Лобби отрисовано визуально.

## Не реализовано / на будущее
- Тайм-банк на ход (сейчас ход не истекает по времени).
- Докупка фишек (rebuy) при нуле; сейчас игрок с 0 просто вне раздач.
- Турнирный режим/рост блайндов. Пока только кэш-игра.
- Для продакшена в Telegram: хостинг с HTTPS + регистрация Mini App в BotFather.
  Состояние столов в памяти процесса (при рестарте теряется).
